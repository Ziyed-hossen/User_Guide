Key Features
============

Features
--------

- Monthly Data available at enterprise as well as hotel level
- Ability to filter the dashboard with more relevant search criteria
- Interactivity like hovering capability and clicking action filters
- One-page property report
- Ability to download reports in CSV and PDF format

Note: As IHG data from HRI is consumed, it takes time for revenue to actualize due to reward redemptions. We pull data on the 5th but data continues to fluctuate with a small variance of generally less than $1000.
